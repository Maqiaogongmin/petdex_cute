# Aka Pilot Codex Pet

这是一个 Codex 桌面宠物项目。宠物名为 **Aka Pilot**，是一个 chibi / pixel-adjacent 风格的红色驾驶员桌宠。

## 当前版本

当前保留的是最终版本：

- 金棕 / 蜂蜜棕双马尾发色
- 蓝色眼睛
- 红色驾驶服
- 橙红胸部细节
- 绿色领口点缀
- 更可爱、更柔和的小骄傲表情

宠物包位于：

```text
pets/aka-pilot/
├── pet.json
└── spritesheet.webp
```

## 项目结构

```text
pets/aka-pilot/
├── pet.json
└── spritesheet.webp
```

## Petdex 上传

上传到 Petdex 时，请上传 `pets/aka-pilot` 文件夹，或上传一个根目录直接包含 `pet.json` 和 `spritesheet.webp` 的 zip 包。

## 安装

把 `pets/aka-pilot` 复制到 Codex 自定义宠物目录：

```text
${CODEX_HOME:-$HOME/.codex}/pets/aka-pilot/
```

然后重启 Codex，在 Settings -> Appearance -> Pets -> Custom pets 中选择 `Aka Pilot`。
